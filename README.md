# 📚 Révisions Philosophie - Spécial Bac

> **Un outil interactif conçu pour réviser les notions clés, les auteurs et les problématiques du bac de philosophie.**

Ce projet a été créé dans le but d'aider ma classe à préparer l'épreuve de philosophie. Il centralise les leçons importantes et les références d'auteurs incontournables sous une forme claire, structurée et facile à consulter.

---

## 🎓 Pourquoi ce projet ?

Les révisions du bac peuvent être denses. J'ai développé ce site pour offrir :
- Un accès rapide aux leçons fondamentales.
- Une fiche synthétique des auteurs par notions (Vérité, Justice, Science, etc.).
- Un support visuel plus agréable que des notes papier pour réviser n'importe où.

**Partagez-le avec le reste de la classe, et bon courage à tous pour le jour J ! 🍀**

---

## 📖 Contenu des révisions

Le site est organisé en plusieurs sections accessibles depuis la page d'accueil :

### 📝 Les Leçons
* **Leçon 1 — Le Bonheur** : *Dépend-il de nous d’être heureux ?* Analyse des points de vue d'Épicure, Kant, et la distinction entre plaisir et bonheur.
* **Leçon 2 — Le Devoir** : *Pourquoi dois-je faire mon devoir ?* Étude de l'impératif catégorique, de la morale et de la conscience.

### ✍️ Fiche Auteurs
Une base de données classée par thématiques pour retrouver rapidement "qui a dit quoi" :
- **Vérité** (Platon, Protagoras, Einstein...)
- **Justice** (Aristote, Rawls, Marx...)
- **Science** (Popper, Bachelard...)
- **Et bien d'autres...**

---

## 🛠️ Installation et Utilisation

Ce projet est purement **Frontend** (HTML/CSS). Aucune installation complexe n'est requise.

1.  **Télécharger** ou cloner le dépôt.
2.  Ouvrir le fichier `index.html` dans n'importe quel navigateur web (Chrome, Firefox, Safari).
3.  Naviguer entre les leçons via le menu latéral ou les liens de l'accueil.

---

## 📂 Structure des fichiers

- `index.html` : Portail principal pour choisir sa leçon.
- `lecon1_bonheur.html` & `lecon2_devoir.html` : Cours détaillés avec définitions et enjeux.
- `auteur.html` : Répertoire synthétique des philosophes et de leurs thèses.

---

## 💡 Conseils pour les révisions
- Utilisez la section **"Fiche Auteurs"** pour mémoriser des citations précises.
- Relisez bien les **"🧾 Conclusion"** en bas de chaque leçon, elles résument parfaitement le problème philosophique traité.
